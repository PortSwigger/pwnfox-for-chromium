PwnFox For Chromium integrates the core features of the original PwnFox extension into Chromium-based browsers, enabling efficient multi-session web security testing. It facilitates seamless interaction between Chromium and Burp Suite, offering color-coded traffic visualization and streamlined profile management for enhanced testing workflows.

Features

  • Launch Chromium profiles directly from Burp Suite
  • Support for up to 8 isolated browser profiles
  • Color-coded traffic based on profile
  • Persistent configuration and session data

Usage

  1. Open the extension's tab within Burp Suite.
  2. Set the path to your Chromium executable:
    • On most systems, this is typically /usr/bin/chromium, /Applications/Chromium.app/Contents/MacOS/Chromium, or the equivalent for your OS.
    • If using Burp Suite's built-in browser, the Chromium binary can usually be found in the burpbrowser folder.
  3. Optionally, define a custom directory for profile data to keep session files separate across projects.
  4. Click a profile button to launch a Chromium instance tied to that profile. Each profile uses a distinct user data directory, maintaining separate cookies, sessions, and history.
  5. Inside the browser, use the included Proxy Settings Editor to configure proxy behavior. This allows fine-grained control and ensures traffic flows correctly through Burp Suite.
  6. Burp Suite will automatically highlight HTTP traffic in the Proxy tab using the color theme associated with the active profile, making it easy to distinguish traffic sources.
  7. All configurations persist across restarts, so you only need to set things up once.
